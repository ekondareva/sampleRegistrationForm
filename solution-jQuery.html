User Information

This is an HTML form for user registration. User must specify his/her full name, gender, birthday, email, and occupation (job position). All fields except Middle Name and Occupation are obligatory, gender must be specified. Correctness of all fields filling is checked with JavaScript. The spellcheck of the Occupation, if filled, is performed with AJAX request to Wikipedia service.
